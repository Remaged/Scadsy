Permission

This object is the representation of the database "Permission" table. To learn how to use this object please see the Datamapper explanation.

Fields

Below you will find information about the different fields the permission has. You will also find the rules that are connected to each field. If you would want to save an permission to the database make sure that every field abides the rules.

Property Information Type Rules
allowed Whether or not access is allowed to a modules action. The module is described using the following three fields. bool
  • required
module_action_name The name of the modules action String
  • required
  • 'max_size' => 100
  • 'min_size' => 1
module_action_module The name of the module String
  • required
  • 'max_size' => 100
  • 'min_size' => 1
module_action_controller The name of the modules controller String
  • required
  • 'max_size' => 100
  • 'min_size' => 1
group_name The group this rule counts for String
  • required
  • 'max_size' => 100
  • 'min_size' => 1

Relations

Below you will find the foreign_key fields that the module has. For more information about the fields please look in the table above.

Field Relationship type
module_action many-to-one
group_name one-to-many
