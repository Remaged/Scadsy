Installation Instructions

SCADSY is installed in five easy steps:
  1. Unzip the download package.
  2. Upload the SCADSY folders and files to your server. Normally the index.php file will be at your root.
  3. Open your browser and browse to the package location.
  4. Follow the installation script.
  5. Delete the install folder from your server.

That's it! SCADSY is now correctly installed. If you want to develop modules for SCADSY please read the modules section.
