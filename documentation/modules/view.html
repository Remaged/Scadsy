Views

In the previous part we created a controller. This controller showed "Hello world!" without the header and footer. In this part we will create a view, give it certain data and make sure the header and footer load.

To get started:

  1. Go to your "modules/helloworld/views" directory.
  2. In this directory, create a new file called view.php
  3. Paste the html below into the new file.
That's it! You just created your first view. Now we have to change the controller we created to load this view.

<h1><?php echo $title; ?></h1>

<p>Hello world! - <?php echo $author; ?></p>

Open up your HelloWorld controller and replace the 'echo "Hello World!";' with the following code:

<?php 
$data = array();
$data['author'] = "Bob";
$data['title'] = "Hello World Page!";
$this->view('view', $data);
?>

Save the file, go back to your browser and lookup the BASE_URL/index.php/helloworld/helloworld/index again. You should now see your hello world page loaded with the menu.

Ok, so we created a controller, we have a view, but nobody knows that our page exists. Time to add it to the menu and for that we are going to use hooks!
