Controllers

To be able for our plugin to have its own pages we should create a controller. The controller is the connection between the model and the view. We will show how to create these later. First, we will create our own helloworld controller.

Follow the following steps to create your first controller:

  1. Go to your "modules/helloworld/controllers" directory.
  2. In this directory, create a new file called helloworld.php.
  3. Paste the template below into the new file.
That's it! You just created your first controller. If you go to BASE_URL/index.php/helloworld/helloworld/index you should see the hello world message.

<?php  if ( ! defined('BASEPATH')) exit('No direct script access allowed');

class HelloWorld extends SCADSY_Controller{

        public function index() {
                parent::init();

                echo "Hello world!";
        }
}

/* End of file helloworld.php */
/* Location: ./modules/helloworld/controllers/helloworld.php */

In this file we created a new controller that inherits from the SCADSY_Controller. Every controller you create should inherit from this controller. The SCADSY_Controller manages things like security and user validation.

In the index method you will see a call to the SCADSY_Controller::init function. Every page in a controller should always call this function if you want to check the permissions. This function makes sure that the currently logged in user has access to the page.

You will notice that you are missing the header and the footer notices, we will show you how to add these now by creating views.
