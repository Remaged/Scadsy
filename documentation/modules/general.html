Modules

Let's start to talk about modules. Almost every functionality you see within SCADSY comes from a module. The login page is a module, the module page is a module and so on. In this section of the user guide we will explain how you can extend the SCADSY core system by creating your own modules.

This tutorial is composed of a few parts:

  1. Base Structure
  2. Controllers
  3. Views
  4. System Hooks
  5. Other Features
