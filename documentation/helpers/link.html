Link Helper

post_link($page, $text, $data, $succes = '', $fail = '')

The post_link function allows you to easily create a link that submits data onClick. You can also specify what should happen on succes and on failure. Below can find a simple example.

<?php
echo post_link('helloworld/helloworld/index', "Hello World Post Link", array("name" => "hello"), "alert('Succes!');", "alert('Failure!');");
?>

Below you will find the list of the parameters and their meaning.

# Name Function
1 $page The page where the post should be send to.
2 $text The anchor text.
3 $data An array with the data that should be posted.
4 $succes (Optional) The javascript for the succes function. You can acces the link object inside the callback by using the myLink variable.
5 $fail (Optional) The javascript for the fail function. You can acces the link object inside the callback by using the myLink variable.
