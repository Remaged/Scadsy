Partial view Helper

A partial view enables you to define a view that will be rendered inside a parent view.

partial_view($uri)

Creates a partial view using the provided uri-parameter. The call for this partial view will be simular to an AJAX-call. This will result in a view without template.

Example usage:

<h1>Some parent page</h1>
<p>This is content of the parent page</p>
<p>The code below will render a view within this parent page</p>
<?php echo partial_view('/some_module/some_controller/some_action'); ?>

//Results in:
<h1>Some parent page</h1>
<p>This is content of the parent page</p>
<p>The code below will render a view within this parent page</p>
<div data-partial_view="/some_module/some_controller/some_action">Some possible content of the partial_view</div>
