Url Helper

The url helper of CodeIgniter is being extended in Scadsy.

get_uri_segment($segment_number)

Gets a specific segment from the url by the given segment_number.

An uri in Scadsy is defined by module/controller/action/some_parameter. Therefore: module=1, controller=2, action=3, some_parameter=4. Example

<?php
//If, for example, the current url is http://some-site.com/your_module/users/get/5 , then:
echo get_uri_segment(1); //will produce 'your_module'
echo get_uri_segment(2); //will produce 'users'
echo get_uri_segment(3); //will produce 'get'
echo get_uri_segment(4); //will produce '5'
?>

get_uri_segments($number_of_segments)

Simular to uri_string() of CodeIgniter's uri_helper. Rather than giving the entire uri_string, this method allows you to only retrieve a certain number of segments.

<?php
//If, for example, the current url is http://some-site.com/your_module/users/get/5 , then:
echo get_uri_segments(1); //will produce '/your_module'
echo get_uri_segments(2); //will produce '/your_module/users'
echo get_uri_segment(3); //will produce '/your_module/users/get'
echo get_uri_segment(4); //will produce '/your_module/users/get/5'
?>

site_action_uri($action)

This method will create an uri based on the current module, controller and the provided action-parameter. Example

<?php
//If, for example, the current url is http://some-site.com/your_module/users/get/5 , then:
echo site_action_uri('delete'); //will produce '/your_module/users/delete'
?>
