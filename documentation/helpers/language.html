Language Helper

__($text, $params = '')

The __ function allows you to translate certain parts of your template to the currently active language. You can use it in the following way:

<?php 
echo __("Hello world!"); 
?>

Hello world will now be translated to the hello world version of the currently active language. The __ function also supports string formatting so the following code is allowed:

<?php
echo __("I am using %s!", "SCADSY");
?>

The above code will output: "I am using SCADSY!".
