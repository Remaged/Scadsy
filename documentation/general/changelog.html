Version Alpha 0.2

Release Data: Not Yet Released

  • Core Changes
    • All files should be named in lower case.
    • Base system is rewritten to use the new datamapper.
    • The design was revised and it is now required to add an image to a main menu item.
    • The permission system has been reworked. It is now possible to create your own groups and set its permissions.
  • New Features
    • You can now extend existing forms from within modules using the Form Manager and the modules_loaded hook.
    • Datamapper ORM Library is now supported.
    • Added the post_link() function for auto generating ajax post links.
    • You can now show notifications using the Notification Manager and the pre_notifications_generate hook.
    • It is now possible to access the link object in the post_link helper javascript callbacks by using the myLink variable.
    • Modules can now update if the protocol for updating is implemented. Click here for more information.
    • SCADSY now allows internationalization by using the Language Helper and Module Features.
    • It is now possible to add widgets to the dashboard by using the new Dashboard Manager and the pre_dashboard_generate hook.
  • Bug Fixes
    • Sessions are now stored in the correct database while using the enterprise version.
    • Modules are now removed from the database if they are removed from the file system.
