The Menu Manager

The menu manager is responsible for generating the correct menu. You can get access to the menu manager from within the pre_menu_generate hook.

Adding menu items

Here is a simple example showing how you can add a menu items:

<?php
$menu_manager->add_menu_item('helloworld/helloworld/index', 'Hello World');
?>

The above code will generate a new menu item with the label "Hello World" that links towards 'helloworld/helloworld/index'. Since version 0.2 menu items are required to have an icon. Each menu item should have 3 different icons: one as default, one when the menu item gets hovered over and one when the menu item is active. Menu items will be automatically loaded if you follow the following naming convention:

  • modules
    • YOUR_MODULE
      • assets
        • images
          • icon_24.png
          • icon_24_hover.png
          • icon_24_active.png

To create a submenu item you have to call the following function:

<?php
$menu_manager->add_submenu_item('helloworld/helloworld/index','helloworld/helloworld/sub', 'Sub menu');
?>

The above code will generate a new submenu item below the 'helloworld/helloworld/index' menu item.

Getting the menu items

Here is a simple example of getting the html for the menu:

<?php
$menu_manager->get_menu();
?>

If you would want to get the menu items instead of the html you have to call the function with FALSE as parameter:

<?php
$menu_manager->get_menu(FALSE);
?>
