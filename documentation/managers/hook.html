Hook Manager

The Hook Manager is responsible for setting and executing hooks. It's a static class so you can access it from everywhere.

Adding a hook

To hook a function into an existing hook you can use the following piece of code:

<?php
    Hook_manager::add_hook('HOOK_NAME', 'FUNCTION_NAME');
?>

So for example, if you would want to add something to the pre_menu_generate hook you could use the following code:

<?php
Hook_manager::add_hook('pre_menu_generate', 'hello_world_pre_menu_generated');

function hello_world_pre_menu_generated($menu_manager) {
    echo "Hello!";
}
?>

The function is now registered and will be called when the hook triggers

Available hooks

The following hooks are currently available:

Name Purpose Function parameter Example function
pre_menu_generate Add menu items to the menu Menu Manager helloworld_pre_menu_generate($menu_manager)
post_menu_generate Change the menu html after it is generated HTML helloworld_post_menu_generate($html)
pre_scripts_header_generate Add javascript files to the header Template Manager helloworld_pre_scripts_header_generate($template_manager)
post_scripts_header_generate Change the javascripts html after it is generated HTML helloworld_post_scripts_header_generate($html)
pre_scripts_footer_generate Add javascript files to the footer Template Manager helloworld_pre_scripts_footer_generate($template_manager)
post_scripts_footer_generate Change the javascripts html after it is generated HTML helloworld_post_scripts_footer_generate($html)
pre_stylesheets_generate Add stylesheet files to the header Template Manager helloworld_pre_stylesheets_generate($template_manager)
post_stylesheets_generate Change the stylesheets html after it is generated HTML helloworld_post_stylesheets_generate($html)
pre_notifications_generate Add notifications to the header Notification Manager helloworld_pre_notifications_generate($notification_manager)
post_notifications_generate Change the notifications html after it is generated HTML helloworld_post_notifications_generate($html)
pre_dashboard_generate Add widgets to the dashboard Dashboard Manager helloworld_pre_dashboard_generate($dashboard_manager)
post_dashboard_generate Change the widgets html after it is generated HTML helloworld_post_dashboard_generate($html)
modules_loaded Execute script after all modules have been loaded / Add fields to existing forms Form Manager helloworld_pre_form_fields_generate($form_manager)
