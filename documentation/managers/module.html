Module manager

The module manager is responsible for handling everything concerning the modules. The module module that comes with SCADSY uses this manager to load, install, disable and enable modules. The module manager gets loaded by default and is accessible through the main object.

Managing modules

Installing a module

To install a new module that is, for example, in the "modules/helloworld" directory, we could write the following code:

<?php
$SCADSY =& get_instance();
$SCADSY->module_manager->install_module("helloworld");
?>

Or if we would want to call the module manager from within a controller we could use the following code:

<?php
$this->module_manager->install_module("helloworld");
?>

Uninstalling a module

To uninstall the module we just installed, from within a controller, we could write the following code:

<?php
$this->module_manager->uninstall_module("helloworld");
?>

Refreshing a module

The system only checks your modules actions on install. If you would like to add new actions after the module is installed you can use the refresh method. This method searches for new actions inside your module and adds them to the system.

<?php
$this->module_manager->refresh_module("helloworld");
?>

Searching for new modules

Searching for new modules and adding them to the database happens in one simple function. To call this function you can write the following code:

<?php
$this->module_manager->add_new_modules();
?>
