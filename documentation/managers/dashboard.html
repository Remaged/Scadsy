Dashboard Manager

The dashboard manager is responsible for generating the widgets on the dashboard. You can access the dashboard manager from within the pre_dashboard_generate hook.

Adding a widget

Here is a simple example showing how you can add an widget:

<?php
$dashboard_manager->add_widget("helloworld/helloworld/index");
?>

Getting the widgets

Here is a simple example of getting the html for the widgets:

<?php
$dashboard_manager->get_widgets();
?>

If you would want to get the widget items instead of the html you have to call the function with FALSE as parameter:

<?php
$dashboard_manager->get_widgets(FALSE);
?>
