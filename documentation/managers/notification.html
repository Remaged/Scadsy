The Notification Manager

The notification manager is responsible for generating the notifications. You can access the notification manager from within the pre_notifications_generate hook.

Adding a notification

Here is a simple example showing how you can add an notification:

<?php
$notification_manager->add_notification("succes", "You added a notification!");
?>

There are currently three supported notification types:

Type Usage
succes Use this type when you want to send a succes notification.
update Use this type when you want to send a update notification.
error Use this type when you want to send a error notification.

Getting the notifications

Here is a simple example of getting the html for the notifications:

<?php
$notification_manager->get_notifications();
?>

If you would want to get the notification items instead of the html you have to call the function with FALSE as parameter:

<?php
$notification_manager->get_notifications(FALSE);
?>
