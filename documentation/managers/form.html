Form Manager

The form manager is responsible for adding extensibility to existing forms. Lets say that you would want to add an extra field to the login page. You don't want to alter the code of the existing form because that could break things. Instead you are able to add a new field to the login form by using the form manager. You can access the Form Manager through the modules_loaded hook.

Adding a field to a existing form

Using the following piece of code we can add a field to a form:

<?php
Hook_manager::add_hook('modules_loaded', 'helloworld_modules_loaded');
function helloworld_modules_loaded($form_manager) {
    $form_manager->add_fields('helloworld_form',array(
        "<label for='test'>Test</label><input type='text' name='test'/>"
    ),"helloworld_form_callback ");
}

function helloworld_form_callback ($postdata){
    /* Handle the post here, for example: */
    echo $postdata['test'];
}
?>

We just added an input field to the hello world form that calls back to the helloworld_form_callback function.

Allowing expansion to your own forms

Allowing others to expend your own forms is really easy. The only thing you have to include in your form is the following code:

<?php
echo form_extra("your_own_unique_form_identifier");
?>

People will now be able to add fields to your form using the "your_own_unique_form_identifier" identifier.
