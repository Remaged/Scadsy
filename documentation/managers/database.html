Database Manager

Sometimes, you want to access the database to preform actions the DataMapper doesn't support. Within SCADSY you access the database through the Database Manager. The Database Manager is a static class so you can call it from anywhere in your module.

To get the database you have to call the following function:

<?php
$database = Database_manager::get_db();
?>

Using Active Record you can make chances to the database.
